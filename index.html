Projects

01

Bridge Constructor

In this project, I tackled the problem of constructing a bridge over a river with varying depths. Given the depth of the river at different indices, supports can be installed at any index with a length equal to the depth at that point. The challenge is to minimize the total length of supports needed, ensuring that the distance between the indices of two supports does not exceed a given value. This project includes a visualization to demonstrate the workings of dynamic programming and brute force methods, providing an insightful comparison of both approaches.

Chef Help

Chef Help is a responsive recipe finder app built with React and Tailwind CSS. The app fetches a variety of recipes from an API based on user search queries and includes features such as saving favorite recipes for easy access. With a focus on user experience, Chef Help offers a smooth, seamless interface across devices, and allows users to explore and save their favorite dishes with ease.

News Aggregator

This project is a dynamic news aggregator that displays the latest news from various categories, allowing users to easily toggle between different categories. Key features include category filtering, responsive design, and real-time updates.
